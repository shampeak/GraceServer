## Grace\Server

## 说明

服务层容器
包括

- 基础类             Grace/Base
- 服务容器           Grace/Server
- Cookie类           Grace/Cookies
- 数据库访问         Grace/Db
- 请求参数           Grace/Req
- 信息流             Grace/Wise

## 安装

稳定版本 1.0

composer require grace/server 1.0
OR
下载打包好的文件[下载](https://github.com/shampeak/GraceServer/archive/master.zip)

## 使用

- Grace/Base

- Grace/Server

- Grace/Cookies

- Grace/Db

- Grace/Req

- Grace/Wise

- Grace/Cache
