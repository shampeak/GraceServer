## setPrefix
#### 设置cookie的默认过期时间，不建议使用该方法 ####

示例
```
建议到 /Application/Config/Cookies.php 文件中配置

return [

    'prefix'   => 'GraceEasy',         // cookie prefix 前缀
    'securekey'=> 'ekOt4_Ut0f3XE-fJcpBvRFrg506jpcuJeixezgPNyALm',     // encrypt key   密钥
    'expire'   => 36000,     //超时时间

];

```
